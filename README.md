# BoliNews 

API que usa el sitio 
- https://noticias.zetita.online

Que muestra el sitio?
-
Resumenes automaticos humoristicos hechos por IA de los titulares que hay en los periodicos digitales 


Las noticias no tienen que ser aburridas

 


Recursos usados
- 

- API gemini (el que hace los resumenes)
- Python 3.10+
- Flask Framework
- FlaskRestful
- SQLAlchemy 
- JWT
- lxml (para webscraping)
- Mysql 
- Aiven.io(para el deploy de la db capa gratuita)
- Vercel (para el deploy del sitio y esta api)
- Github actions (cronjobs)
